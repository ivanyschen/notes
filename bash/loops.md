# Loops

## while

```bash
while [<some test>]
do
	<commands>
done
```

## until

```bash
until [<some test>]
do
	<commands>
done
```

## for

```bash
for var in <list>
do
	<commands>
done
```

- `a_list='item1 item2 item3'`a list with `item1`, `item2`, `item3`
- ranges can be used: `{1..5}`, `{10..0..2}`

## Select

```bash
select var in <list>
do
	<commands>
done
```
