# If & CaseStatements

## Test

- Different from Python, If a test is **true**, **0 (zero)** is return and vice versa.

## If 

```bash
if [<some test]
then
	<command_a>
elif [<some test]  # can have 0 or multiple of them
then
	<commands_b>
else
	<commands_c>
fi
```

## Case

```bash
case <variable> in
	<pattern 1>)
		<commands_1>
		;;
    <pattern 2>)
    	<commands_2>
    	;;
esac
```
